# Simple Usage

## Start Capturing

1. Click the **Start** button in the upper right corner or the prompt button in the center to enable the proxy function (enabling the proxy will simultaneously set the system proxy to Listen).

2. Open your browser and start browsing. Listen will capture HTTP/HTTPS requests (for HTTPS requests, you need to [install the SSL Certificate](/en/install-cert) first). Double-click a captured request to view its detailed information.

![](img/light/ç®€å•ä½¿ç”¨2.png?version=1.1.0)

3. Right-click a request to perform various operations on it.

![](img/light/ç®€å•ä½¿ç”¨3.png?version=1.1.0)

4. Click the **"Collection Toggle"** button in the bottom left corner to open or close the Collection panel. The Collection panel categorizes captured requests, allowing you to quickly filter and locate them. ## Construct Requests

1. Switch to the **"Send Request"** tab in the sidebar. Manually enter request information, or right-click an existing request in the list and select **"Edit & Resend"** to modify its details.
2. Click the **"Send"** button in the upper right corner to simulate sending the request (for HTTPS requests, you need to [install the SSL Certificate](/en/install-cert) first). Once the server responds, you can view the details in the response area below.

![](img/light/ç®€å•ä½¿ç”¨5.png?version=1.1.0)

3. Click the **"+"** button on the top tab bar to add a new request tab.
4. Right-click on the tab navigation bar to perform various operations. You can save requests; saved requests can be viewed, re-edited, and re-sent from the Collection area on the left.

![](img/light/ç®€å•ä½¿ç”¨6.png?version=1.1.0)
