# 简单使用

## 开始抓包

1. 点击右上角开始按钮或者中间的提示按钮，开启代理功能（开启代理的同时会将系统代理设置为Listen）。

![](img/light/简单使用1.png?version=1.1.0)

2. 打开浏览器开始浏览网页，Listen即可捕获到HTTP/HTTPS请求（对于HTTPS请求，需要先[安装SSL证书](/zh/install-cert)）。双击捕获到的请求，可以查看请求的详细信息。

![](img/light/简单使用2.png?version=1.1.0)

3. 右键请求，可以对请求进行各种操作。

![](img/light/简单使用3.png?version=1.1.0)

4. 点击左下角的“集合面板开关”按钮，可以打开或者关闭集合面板，在集合面板里，对捕获到的请求进行了分类，在这里，可以快速对请求就行过滤和定位。另外，点击请求列表右上角的"🔍"按钮，可以对请求进行精确过滤。

![](img/light/简单使用4.png?version=1.1.0)

## 构造请求

1. 在侧边栏将当前界面切换到“发送请求”Tab页面，手动输入请求信息或者在请求列表右键已有请求，选择“编辑 & 重发”，然后修改请求信息。
2. 点击右上角的“发送”按钮，即可模拟请求发送（对于HTTPS请求，需要先[安装SSL证书](/zh/install-cert)）。服务器返回后，可以在在下方的响应区域查看响应信息。

![](img/light/简单使用5.png?version=1.1.0)

3. 点击顶部Tab栏上的“+”按钮，可以新增请求Tab页。
4. 在Tab导航栏上右键，可以对Tab页进行各种操作。可以对请求就行保存，保存好的请求可以在左侧集合区中查看并重新编辑和发送。

![](img/light/简单使用6.png?version=1.1.0)
