Core Features

Full control over network requests to boost development and testing efficiency

Gateway

Block requests from being sent to the server or prevent responses from returning to the client.

Mirror

Replace the request domain with a specified domain, similar to modifying local hosts files.

Mapping

Map request URLs to local file paths or directories for easy debugging.

Rewrite

Automatically modify and replace requests and responses using pre-defined rules.

Breakpoint

Intercept requests and responses for manual modification or inspection.

Scripting

Write JavaScript code to intercept and modify requests and responses with maximum flexibility.

WebSocket

Inspect WebSocket connections and simulate message sending. Intercept messages via Rewrite and Scripting.

API Mocking

Construct arbitrary HTTP/HTTPS requests to test your interfaces easily.

Gateway
The Gateway feature allows you to block or hang requests, preventing them from reaching the server for filtering and control.
Mirror
Mirroring redirects requests by replacing domains. It's more convenient and faster than manually editing your hosts file.
Mapping
Mapping replaces request URLs with local file paths or directories, enabling local mapping of individual files or entire sites.
Rewrite
Rewrite requests before they reach the server or modify responses before they return to the client based on custom rules.
Breakpoint
Pause requests or responses at specific points for manual inspection and modification during the debug process.
Scripting
Use custom JavaScript code to programmatically modify requests and responses for flexible control and complex logic.
WebSocket
Excellent WebSocket support for viewing and searching messages. Send messages directly or intercept them using the Rewrite and Scripting features.
API Mocking
Construct any HTTP/HTTPS request (URL, Method, Header, Body) to test API interfaces. Supports collection storage for reuse.
